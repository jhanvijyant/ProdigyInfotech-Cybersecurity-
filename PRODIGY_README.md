---

## Tasks

### PRODIGY_CY_01 — Caesar Cipher Encryption Tool
- Implemented a Caesar cipher encryption and decryption program
- Supports variable shift values and handles both uppercase and lowercase characters
- **Concepts:** Classical cryptography, substitution ciphers, character encoding
- **Stack:** Python

---

### PRODIGY_CY_02 — Pixel Manipulation & Image Encryption
- Built an image encryption tool using pixel-level XOR operations
- Encrypted and decrypted image files without external libraries
- **Concepts:** XOR encryption, image steganography basics, binary data manipulation
- **Stack:** Python, PIL/Pillow

---

### PRODIGY_CY_03 — Password Strength Checker
- Developed a command-line password strength evaluator
- Assessed length, character diversity, and common pattern detection
- **Concepts:** Authentication security, NIST password guidelines
- **Stack:** Python

---

### PRODIGY_CY_04 — Keylogger (Ethical Research)
- Built a basic keylogger for educational and ethical research purposes only
- Captures keystrokes locally and logs to a file — demonstrates how keyloggers work so defenders can detect them
- **Concepts:** Input capture, malware behaviour analysis, endpoint security
- **Stack:** Python, pynput
- **Note:** Intended strictly for understanding attack patterns and improving defensive detection

---

### PRODIGY_CY_05 — Network Packet Analyser
- Developed a packet sniffer to capture and inspect network traffic in real time
- Displayed source/destination IPs, protocols, and payload previews
- Aligned with Wireshark-style analysis for educational purposes
- **Concepts:** Network forensics, packet analysis, OSI model, TCP/IP
- **Stack:** Python, Scapy

---

## Key Skills Demonstrated

- Python scripting for security tooling
- Cryptography fundamentals (symmetric, XOR, Caesar)
- Network packet capture and analysis
